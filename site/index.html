About the Project

About SiteSpy

SiteSpy is a mobile-first construction project records and cost estimation app built with Expo React Native, Firebase, and Firestore. It helps construction teams capture measurements, estimate material costs, and review project history on any device — even in low-connectivity field conditions.

My Contribution

My Contribution

I built the core navigation structure and dashboard entry point for SiteSpy. SplashScreen.js handles the branded app launch experience. DashboardScreen.js provides the main landing view users see after signing in, with summary cards and quick-access entry points. MainTabs.js defines the tab bar navigation.
