About the Project

About SiteSpy

SiteSpy is a mobile-first construction project records and cost estimation app built with Expo React Native, Firebase, and Firestore. It helps construction teams capture measurements, estimate material costs, organise site records, and review project history on any device — even in low-connectivity field conditions.

My Contribution

My Contribution

I built the core navigation structure and dashboard entry point for SiteSpy. SplashScreen.js handles the branded app launch experience. DashboardScreen.js provides the main landing view users see after signing in, with summary cards and quick-access entry points into their project workflows. MainTabs.js defines the tab bar navigation that moves users between the app's main sections. StatCard.js is the reusable metric card component used across the dashboard.

Reflection

My Reflection

Navigation is the skeleton of a mobile app — if it is confusing or sluggish, users abandon the product. I learned how to structure nested tab and stack navigators cleanly, how to manage the splash-to-dashboard transition without flicker, and how to make a dashboard that communicates app value in the first two seconds. Mobile-first thinking was essential.
